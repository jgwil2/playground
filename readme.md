# Playground

Solving puzzles and exploring type annotations with Python 3.

## Type Hints

Type annotations require mypy:

```
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
mypy .
```

See also: [Python type checking in VS Code](https://stackoverflow.com/questions/45829353/python-type-checking-in-vs-code)
